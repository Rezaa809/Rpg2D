## 📖 Tentang Game

**RPG2D (Bola Peperangan)** adalah game *Role-Playing Game* 2 dimensi klasik dengan gaya visual pixel-art. Jelajahi dunia, kalahkan musuh dalam pertempuran sengit, dan jadilah yang terkuat!

---

## 📥 Download & Instalasi (Android)

Kamu bisa langsung mendownload file APK-nya dari repositori ini:

### **[👉 KLIK DISINI UNTUK DOWNLOAD APK 👈](https://github.com/rhezakahhh/rpg2d/raw/utama/apk%20v1%20(tanpa%20suara)/rpg2d.apk)**

> **Catatan:** Ganti `rpg2d.apk` pada link di atas dengan nama file asli yang ada di dalam folder tersebut jika berbeda.

**Cara Instalasi:**
1. Klik link download di atas.
2. Jika ada peringatan "File mungkin berbahaya", pilih **Tetap Download**.
3. Buka file APK dan pilih **Install**. (Pastikan izin "Sumber Tidak Dikenal" sudah aktif di pengaturan HP kamu).
